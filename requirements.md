## Packages
recharts | Beautiful data visualization for the dashboard
framer-motion | Smooth animations and page transitions
date-fns | Date and time formatting
lucide-react | Tactical icons for the UI

## Notes
- Theme is strictly dark mode, representing a tactical/security environment.
- The Monitoring page uses WebRTC to access the local webcam. It handles denial gracefully by showing a "Signal Lost" state.
- Tailwind config should extend fonts to use `font-display` and `font-mono`.
